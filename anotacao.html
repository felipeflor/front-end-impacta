CSS significa Folhas de Estilos em Cascata Formas de implementação: *inline: CSS aplicado DIRETAMENTE em cada tag HTML *interno/onpage CSS aplicado na página, ou seja, regras CSS criadas desta forma valem para todos os elementos daquela página. *externo CSS desenvolvido em um arquivo de extensão .css, ele é conectado às páginas HTML. Framework: conjunto de recursos, funcionalidades, regras, documentação.
